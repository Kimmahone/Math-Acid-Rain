🌧️ 수학 산성비 게임(Math Acid Rain)

떨어지는 수학 수식을 암산으로 풀며 연산 실력을 키우는 웹 기반 교육용 미니 게임입니다.
HTML, CSS, JavaScript 단일 파일로 구성되어 있어 별도의 설치 없이 브라우저에서 바로 실행할 수 있습니다.

🎮 게임 특징 (Features)

5단계 난이도: 🐢 매우 느림부터 ⚡ 매우 빠름까지 사용자의 수준에 맞게 속도를 선택할 수 있습니다.

다양한 문제 유형: 하, 중, 상으로 나뉜 사칙연산 혼합 계산 문제(총 100문제)가 무작위로 출제됩니다. 모든 문제의 답은 99 이하의 자연수로 떨어지도록 최적화되어 있습니다.

콤보 및 보너스 시스템: 연속으로 정답을 맞히면 추가 점수(Combo)를 얻으며, 10연속 정답 시 생명(❤️)을 추가로 획득합니다.

사용자 친화적 UI: 정답 격파 애니메이션 효과, 반투명 입력창, 오답 시 흔들림(Shake) 효과 등을 제공하여 몰입감을 높였습니다.

🚀 실행 방법 (How to Play)

이 저장소의 코드를 다운로드(Clone) 하거나, index.html 파일을 직접 다운로드합니다.

다운로드한 index.html 파일을 웹 브라우저(Chrome, Edge, Safari 등)로 더블 클릭하여 엽니다.

원하는 난이도를 선택한 후, 떨어지는 수식의 답을 하단 입력창에 적고 Enter 키를 눌러 블록을 파괴하세요!

🛠️ 사용된 기술 (Tech Stack)

HTML5 & CSS3 (Animations, Flexbox)

Vanilla JavaScript (ES6+)

Tailwind CSS (CDN 방식으로 적용하여 스타일링)

Lucide Icons

📝 문제 난이도 구성

하 (초록 배지): 단순 덧셈, 뺄셈, 곱셈, 나눗셈

중 (주황 배지): 사칙연산이 2~3개 섞여있는 식 및 괄호 포함

상 (빨강 배지): 덧셈, 뺄셈, 곱셈, 나눗셈이 모두 섞여있는 복합 계산식
